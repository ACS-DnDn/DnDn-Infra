# Prod Environment Skeleton

이 디렉터리는 `prod` 환경용 GitOps 골격입니다.

현재는 아직 앱 레포 구현과 운영 기준이 완전히 고정되지 않아, 실제 root app과 child application은 추가하지 않았습니다.

현재 목적은 아래 두 가지입니다.

- `dev`와 같은 구조를 미리 맞춰두기
- 나중에 `prod`용 root app, child app, values를 추가할 자리만 먼저 확보하기

추후 추가 예정:

- `root/`
- `apps/dndn-web`
- `apps/dndn-api`
- `apps/dndn-worker`
- `apps/dndn-report`
- `apps/dndn-hr`
